# WSA Prada Reddit Analysis

This repository contains the source code for an academic university project for the course Web and Social Media Search and Analysis.

The project uses the Reddit API through PRAW in read-only mode to collect publicly available Reddit submissions and comments related to discussions around *The Devil Wears Prada 2*.

The collected data will be used for:
- social network analysis;
- user-user reply network construction;
- community detection;
- sentiment and emotion analysis;
- named entity recognition;
- aggregate-level visualizations and reporting.

The application does not post, comment, vote, moderate communities, send messages, or interact with Reddit users. It only reads publicly available Reddit content for non-commercial educational research.

Credentials and API secrets are not stored in this repository.
